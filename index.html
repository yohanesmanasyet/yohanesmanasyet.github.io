OCR Project for Indonesian Election Results 2024

Extracting Data from 2024 Election Results Form



Description

This project focuses on using Optical Character Recognition (OCR) to extract data from Indonesian election forms for the 2024 elections. The process involves two main steps: first, using object detection with the Detectron model to identify and create bounding boxes around numbers on the forms. Second, a Transformer-based model is employed to perform OCR on the detected areas, accurately extracting handwritten and printed text. This approach enhances the accuracy and efficiency of data processing, supporting comprehensive analysis and reporting. This project was also part of a competition, showcasing innovative solutions in data processing and electoral transparency.

H1N1 and Seasonal Vaccine Prediction Model

A Machine Learning Approach for Influenza A (Seasonal) and H1N1 Vaccine Probability.

Description

This project focuses on predicting the likelihood of individuals receiving vaccines for H1N1 and seasonal influenza. The evaluation results are shown through ROC curves for both vaccines, where the AUC (Area Under the Curve) values indicate the model's effectiveness. For the H1N1 vaccine, the AUC is 0.8394, demonstrating good predictive performance. Similarly, the seasonal vaccine model achieves an AUC of 0.8585, indicating slightly better accuracy. These results suggest that the model is effective in distinguishing between vaccinated and non-vaccinated individuals, providing valuable insights for vaccination campaigns and public health strategies.

Text Clustering with TF-IDF and KMeans

Analyzing and Categorizing Online News Articles

Description

This project focuses on Natural Language Processing (NLP) techniques to cluster text from a collection of online news articles. Using TF-IDF for feature extraction, the text data is processed and analyzed to identify underlying patterns. KMeans clustering is then applied, categorizing the articles into three distinct topics: Government, COVID-19, and Natural Disasters. This approach allows for efficient organization and analysis of large datasets, providing valuable insights into prevalent themes in news coverage.
